# She Can Foundation Website

A modern responsive contact form website created for She Can Foundation to support and empower women through technology and innovation.

## 🌟 Features

* Responsive Design
* Modern UI/UX
* Interactive Contact Form
* Clean and Professional Layout
* Node.js Backend Integration

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js

## 📂 Project Structure

```bash
She-Can-Foundation/
│
├── public/
│   ├── index.html
│   ├── style.css
│   └── script.js
│
├── server.js
├── package.json
├── package-lock.json
├── README.md
└── .gitignore
```

## 🚀 How To Run Project

1. Install dependencies

```bash
npm install
```

2. Start server

```bash
node server.js
```

3. Open browser

```bash
http://localhost:3000
```
